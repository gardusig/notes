## 📌 2. What to use where

| Lane | Reach for |
| --- | --- |
| [Sardine, egg & rice pan](sardine-rice-eggs.md) | **Tempero baiano** + black pepper; optional caldo in the rice water. |
| [Mexidão](meat/mexidao.md) | Bay, pepper, garlic/onion powder; **not** tempero baiano as the main note unless you want a fusion experiment. |
| [Bean soup](soup/bean.md) | Bay leaves, pepper; pinch **cumin** or **tempero baiano** after blend. |
| [Vegetable soup](soup/vegetable.md) | Pepper; curry / turmeric if the pot is going that direction. |
| [Pizza](italian/pizza/README.md) variants | **Oregano** on sauce; margherita stays basil-forward. |
| [Pasta](italian/pasta.md) | Salt, pepper; oregano only if the sauce is dry/tomato-heavy. |

**Tempero baiano** — typical blend includes paprika, turmeric, oregano, cumin, and chili; good default when you want **Brazilian savor** without opening five jars. Start **light** (½–1 tsp per pan) and adjust; it already carries salt in many brands—taste before adding more salt.

**Other suggestions to add over time**

- **Smoked paprika** — if you want smoke without buying a second “regular” paprika jar; otherwise sweet paprika above is enough.
- **Fine salt** (refill box) — keep separate from the spice table; every cook uses it.
- **Dried parsley** — optional garnish for pasta / pan finishes; low priority if you buy fresh sometimes.
