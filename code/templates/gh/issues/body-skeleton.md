# GitHub issue body (paste and fill)

**Title:** use [`../../shared/title.md`](../../shared/title.md) — **no emoji** in the issue title.

**Body:** use **`##` section headings** with **one leading emoji** each; keep paragraphs and bullets mostly plain.

**Tables:** [`../../doc/table.md`](../../doc/table.md) or comparison matrices from [`../../doc/readme/TABLE_COMPARISON_TEMPLATE.md`](../../doc/readme/TABLE_COMPARISON_TEMPLATE.md).

**Diagrams (optional):** small fenced mermaid per [`../../diagrams/README.md`](../../diagrams/README.md) and [`../../doc/diagram-conventions.md`](../../doc/diagram-conventions.md); link to **`docs/`** for large graphs.

```markdown
## ⚡ Summary
- **Problem / ask:** …
- **Proposed outcome:** …

## 📋 Context
- …

## ✅ Acceptance criteria
- [ ] …
- [ ] …

## ⚖️ Options considered
- **…** — …

## ⚠️ Risks / open questions
- …
```
