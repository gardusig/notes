# 🔁 Sleep (night routine)

**Target window:** 22:30 → 07:00  

**Upstream:** [Routine hub](README.md)

## 📌 Wind down (22:00–22:30)

1. Light stretch  
2. Hygiene + lotion  
3. Relaxation — keep screens and stimulating content **out of bed**; dim lights; same order nightly helps the brain hand off  
4. Prepare sleep setup (temperature, darkness, phone away from pillow)  

## 📌 Screen and stimulation control

- Prefer **hard stop** on work and social feeds before wind-down.  
- If you use passive entertainment to decompress, **time-box** it so it does not eat into sleep debt.  

## 😴 Sleep

- Aim for **≥ 8 h** in the target window when life allows.  
- **No stimulating inputs** after wind-down starts (news, arguments, bright task lists) — park them for morning.  

If sleep is repeatedly broken, treat it as a **P0 / health** issue before optimizing task throughput — see [`../rules/governance.md#when-emotional`](../rules/governance.md#when-emotional) and [`../rules/priority.md`](../rules/priority.md).
