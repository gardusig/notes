# Data

Many **template** or **docs-first** repositories have **no** application database—state is **files in git** plus optional local tooling under **`.gitignore`**.

## Local-only artifacts

- Editor or agent scratch paths (e.g. **`.cursor/`** research files)—document lifecycle in **`README`** or **`CONTRIBUTING`** if your team uses them.
