# Video games — runbook

**This folder is for:** **video games**—what you play, what you might play, and **reference** (2D vs 3D focus, emulation). Puzzle-first titles you track as *puzzles* rather than “video games” can live under **[puzzles/](../puzzles/README.md)** instead.

## What’s in this folder

| File | Purpose |

| **[emulation.md](emulation.md)** | Emulators, ROMs policy, setup notes. |
| *(optional `.md` you add)* | e.g. `backlog.md`, `now-playing.md`—add a row here when you create one. |

## How to use this folder

1. **Reference:** Edit **2d.md**, **3d.md**, and **emulation.md** when your library or setup changes; they are the long-lived catalogs.
2. **Activity:** Add short files for “what I’m playing” or inbox lists if you want; register each file in the table above.
3. **Inbox → active:** Capture titles you might start, then promote a few when you actually play.

## Suggested habits *(edit to match your life)*

- **Backlog:** games you own or want and might start.
- **Active:** one or two you’re actually playing.
- **Evergreen:** quick sessions you return to often.
