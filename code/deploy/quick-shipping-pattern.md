# Quick shipping pattern

**Upstream:** [`README.md`](README.md)

1. Draft a concrete issue with [`../craft-issue/creativity/create-engineering-task.md`](../craft-issue/creativity/create-engineering-task.md).
2. Refine plan and ambiguity until verify steps are explicit.
3. Use issue triage before creating duplicate threads.
4. Implement in slices and run local verification repeatedly.
5. Calm-read diffs and test output before opening or updating a PR — [`../quality/review-calm-read.md`](../quality/review-calm-read.md).
