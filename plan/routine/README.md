## 🌊 Core flow (summary)

1. Night routine → sleep  
2. Morning routine  
3. Work — onboarding and task picking per [`../rules/priority.md`](../rules/priority.md)  
4. Nutrition and hydration  
5. Repeat  

## ☀️ Mandatory daily rules

- Brush teeth after every meal  
- Morning drink (e.g. juice you use as a habit anchor) every morning  
- Dinner ~19:00  
- Hydration through the day  
- Lunch ~13:00  
- Morning routine before work  
- Night routine before sleep  
- Sleep protected (non-negotiable)  
- Top task defined before starting work — [`../../README.md#daily-capture-and-modes`](../../README.md#daily-capture-and-modes)  
- Water before any meal
