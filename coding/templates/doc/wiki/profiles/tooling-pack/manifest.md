# Wiki manifest (tooling / meta repository)

Optional **`docs/modules/`** (or a single **`docs/README.md`**) for human navigation: separate **bootstrap** phase with user confirm before bulk creates.

## Typical paths

- `docs/README.md` — top hub
- `docs/modules/<domain>/README.md` — one per **top-level concern** (e.g. `ci`, `release`, `templates`), not per leaf file
- `.cursor/rules/` — short agent/human guardrails (stack-specific)
