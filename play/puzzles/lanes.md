# Completion puzzles *(non-ranked)*

**Purpose:** puzzle types **without** a rating. Progress is **finishing** grids or issues—accuracy still matters, but there is **no global ladder**. Track **lifetime totals** per type and optional **weekly throughput** so rotation stays honest.

## Lane catalog

| Lane                | One-line description                                                                                                                |
| ------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| **Patches**         | Fill a grid with **variable-size rectangles** (region packing / tiling style puzzles).                                              |
| **Zip**             | Draw a path (**snake**) through cells in **sorted order** (Zip-style rule).                                                         |
| **Sudoku**          | Classic **9×9** (or variant) digit placement under standard constraints.                                                            |
| **Tango**           | Place **two symbols** (e.g. 0/1) in a grid so **row/column / local rules** forbid bad repetition (LinkedIn-style “Tango” logic).    |
| **Queens**          | Place **one queen (or marker) per row/column/region** per puzzle rules (queens / “one per region” grids).                           |
| **Logic**           | At least **one** general logic puzzle (newspaper, app, or print).                                                                   |
| **Word (optional)** | Crossword, cryptic, connections-style, or other **word** puzzles when you want variety—**no** pressure to hit every subtype weekly. |
