## 📜 Conventions

- **One cook = complete shopping** for that run (no “half a juice pack” as a planning unit; scale the recipe or the pack intentionally).
- **Eating setup:** saucy mains → large spoon + deep bowl; batch weeks → seven matching containers for lunch soup (see [vegetable soup](soup/vegetable.md) portioning).
- **Cross-hub:** all **juice / coffee / tea / soda** runbooks → [`drink/README.md`](../drink/README.md).
