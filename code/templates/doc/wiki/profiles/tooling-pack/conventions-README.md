# Conventions (docs + automation)

## Authoring

- **[`../../../../markdown-and-docs-conventions.md`](../../../../markdown-and-docs-conventions.md)** — indexes, tables, explanation pages
- **`## Contents`** — numbered nested lists; **no** markdown tables for pure navigation indexes
- **Bulk or destructive changes** — confirm with the team before apply

## Layout

- **Public** — user-facing scripts, templates, and runbooks under agreed roots (`templates/`, `scripts/`, etc.).
- **Internal** — shared libraries or helpers; document boundaries in module READMEs, not only in chat.
