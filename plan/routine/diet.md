# 🥗 Diet

Meal times, defaults, and flexibility. Hydration moments that pair with meals live in [hydration.md](hydration.md).

**Upstream:** [Routine hub](README.md)

## 📌 Meal times

- **Dinner ~19:00**  
- **Lunch ~13:00**  

## 📌 Rules

- Avoid sugar / desserts as a default  
- Dinner: **protein + carbs**  
- Lunch: **vegetable soup** (default)  

## 📌 Hunger

1. **Water first** (see [hydration.md](hydration.md))  
2. If still hungry: fruit **before** meals  

## 📌 Meal protocol

| Step | Action |
| --- | --- |
| Before | Large mug of water |
| After | Mouthwash → brush teeth |
